AdPlayer广告播放终端系统使用说明书

1. 安装
本软件为绿色版，不需安装，直接解压至任意目录。

2. 依赖
本软件基于微软.net框架开发，需要.net framework 2.0或更高版本运行库支持。
2.0版本下载地址：http://www.microsoft.com/downloads/zh-cn/details.aspx?FamilyID=0856eacb-4362-4b0d-8edd-aab15c5e04f5&displaylang=zh-cn

3. 视频解码器
如需扩展本软件所支持的视频格式，需要安装相应解码器。
可安装K-Lite code Mega Code Pack解码器，下载地址：http://www.onlinedown.net/soft/15531.htm

4. 如何播放
运行根目录下主程序Router.exe，点击右上角"播放广告"按钮，开始播放。全屏播放过程中可按ESC见返回程序界面（PowerPoint文件播放过程中不可中断，可通过点击鼠标加快页面播放速度）。

5. 如何配置
配置信息分3部分：A.播放文件列表配置，B.通知显示配置，C.幻灯片时间配置， D.屏幕选择
A.播放文件列表配置
    可通过界面方式和直接修改xml配置文件方式操作。建议通过界面方式操作，避免配置信息错误。
    配置界面分上下两部分，上方为树形结构，显示配置文件内容，包括日期时间段、每天时间段、播放文件路径
    下方为选中配置项的具体内容，可手动修改，并更新到上方树形结构中
    修改完成后，点击右下保存按钮，写入配置文件中
B.通知显示配置
    （通知功能未支持）通知显示字体、字号、颜色、背景色、滚动速度
C.幻灯片时间配置
    幻灯片放映时，每页停留时间可通过界面设置，统一为同一时间间隔，否则，将按照配置文件中描述，区别对待
D.屏幕选择
    如有多屏幕显示，可选择在哪一屏幕播放广告，屏幕编号参见Windows显示设置：桌面-属性-设置-鉴别

6. 升级更新
本软件支持优盘直插自动更新。将升级文件update.zip（由官方提供，已加密）保存到任意优盘、移动硬盘根目录中，在软件运行过程中插入连接主机，待升级完成后软件会自动重启，如需继续播放需人工操作。

7. 支持的媒体文件格式
本软件通过安装相应的解码器，可支持大部分视频文件格式。欲验证是否支持播放某一特定视频文件，可通过启动“开始-程序-附件-娱乐-Windows Media Player”，来测试播放该视频文件。若Windows Media Player可顺利播放，则本软件也可以支持播放，反之亦然。
